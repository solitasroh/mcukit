# Phase 2 Convention Eval Prompt - Keyword Trigger Accuracy

User request to define coding conventions for a React/TypeScript project.
The project uses Next.js 14 with App Router, Tailwind CSS, and plans
to follow a feature-based folder structure.

Test keyword trigger accuracy for the phase-2-convention skill activation.
The skill should detect the "convention" and "coding style" intent,
provide convention templates, and generate deliverables including
CONVENTIONS.md, naming rules, and folder structure documentation.

Context: Project level is Dynamic. Phase 1 (Schema/Terminology) has
been completed with a glossary at docs/01-plan/terminology.md.

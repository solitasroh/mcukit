# PM Discovery Eval Prompt - Keyword Trigger Accuracy

User request for market analysis and opportunity discovery for a SaaS
feature called team-collaboration. The user wants to understand the
target market, competitor landscape, and value proposition before
starting development.

Test keyword trigger for pm-discovery skill activation when the user
mentions "market analysis", "opportunity discovery", and "SaaS feature"
as intent signals. The skill should orchestrate the PM Agent Team with
4 specialized agents running in parallel to produce a comprehensive PRD.

Context: Project level is Dynamic. Agent Teams environment variable
CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1 is set. No existing PRD or
Plan documents for team-collaboration feature.
